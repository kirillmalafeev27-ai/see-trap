## Как запустить через интернет (самый простой путь)

### Что получится
- Учитель открывает: `https://<ваш-домен>/teacher.html`
- Ученик открывает: `https://<ваш-домен>/student.html`
- Всё синхронизировано в реальном времени.

### Вариант 1 (рекомендуемый): Render (24/7)

1. Загрузите папку `german-sea-trap-online` в GitHub как репозиторий.
2. Зайдите на https://render.com и войдите.
3. Нажмите `New +` -> `Web Service`.
4. Выберите ваш GitHub-репозиторий.
5. Render сам увидит `render.yaml`.
6. Нажмите `Create Web Service`.
7. После деплоя получите ссылку вида:
   `https://german-sea-trap-online.onrender.com`
8. Откройте:
   - Учитель: `https://.../teacher.html`
   - Ученик: `https://.../student.html`

### Gemini ключ (опционально)
- В Render откройте сервис -> `Environment`.
- Добавьте переменную:
  - `GEMINI_API_KEY=ваш_ключ`
- Сохраните и дождитесь перезапуска.

### Вариант 2 (быстрый, но временный): Tunnel
- Подходит для быстрого теста без деплоя.
- Запускаете сервер локально и поднимаете tunnel (ngrok/cloudflared).
- Минусы: компьютер должен быть включен, адрес может меняться.
